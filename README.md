# PizzaTime

Would You like to taste delicious pizza?<br> Don't waste Your time, order it right now using PizzaTime app!<br><br>

# Quick Start

From the `client` directory:

```bash
docker build -t pizzatime-fe .
```

From the project root:

```bash
docker build -t pizzatime-be .
docker compose up
```

Once all services are running, the application will be available at:
http://localhost:8000/

To stop and remove all running containers:

```bash
docker compose down
```
